# assets/

Developer assets that are not part of the shipped frontend bundle.

## desktop-preview-icon.png — TEMPORARY

`desktop-preview-icon.png` is a **temporary, placeholder** 1024×1024 app icon
(calm teal, non-clinical, no text/logos) created on 2026-07-12 only to enable
native `tauri dev`/`build` locally. The generated icons in `src-tauri/icons/`
come from it.

**Before distribution, replace this with approved release branding** and re-run
`npm run tauri icon assets/desktop-preview-icon.png`.
